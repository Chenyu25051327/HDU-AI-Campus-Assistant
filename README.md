# 杭电AI校园助手

## 一、项目简介

本项目是一个面向杭州电子科技大学学生的校园生活辅助网页，主要使用 HTML、CSS 和 JavaScript 开发完成。

这个项目主要面向对象是刚刚入学的大一新生，为他们在入学时遇到一些不知道该如何解决的问题提供帮助，并可以作为他们获取校园信息的一种途径。

## 二、项目功能

### 1. 校园问答助手

新生们可以输入校园生活相关问题，例如：

- 校园卡丢了怎么办？
- 图书馆怎么预约？
- 快递在哪里取？
- 考试怎么复习？
- 请假怎么申请
- 宿舍空调坏了怎么办？

系统会根据关键词匹配相应回答，并给出校园生活建议。

### 2. 宿舍报修助手

新生输入宿舍楼号、寝室号和故障类型后，系统会自动生成一段报修文本，方便学生向宿管或维修人员描述问题。

### 3. 课程学习建议

用户输入课程名称后，系统会根据课程关键词返回相应学习建议，例如程序设计、高等数学、大学物理等。

- HTML：负责网页结构

## 五、运行方式

本项目不需要安装额外环境，也不需要配置服务器。

运行步骤：

1. 下载或解压项目文件夹。
2. 找到 `index.html` 文件。
3. 双击 `index.html`。
4. 使用浏览器打开网页。
5. 在网页中输入问题或填写信息，即可测试项目功能。

## 七、项目特色

这个项目的特色主要有三点：
1. 内容贴近校园生活  

   问答内容不是简单的通用模板，而是加入了我们杭电学生有可能遇到的实际问题。

2. 操作方式简单  

   项目不需要安装环境，直接打开网页就能运行，有利于课程展示。

3. 体现了 AI 辅助开发过程  

   项目中保留了 AI 辅助生成、人工修改、功能测试和文档整理的过程

## 九、目前不足

这个项目目前还有一些不足：

1. 校园问答主要依靠关键词匹配，不是真正的智能问答；

2. 回答内容还比较固定，不能根据用户问题进行灵活变化；

3. 没有接入真实学校数据库,部分信息仍需要以学校官方通知为准；

4. 页面功能比较基础，还可以继续增加更多模块；

5. 并没有接入API，回答略显死板。
