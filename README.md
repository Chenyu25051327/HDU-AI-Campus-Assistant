# 杭电AI校园助手

## 一、项目简介

本项目是一个面向杭州电子科技大学学生的校园生活辅助网页，主要使用 HTML、CSS 和 JavaScript 开发完成。

项目围绕大学生日常学习和生活场景，设计了校园问答助手、宿舍报修助手和课程学习建议三个功能模块。用户可以通过输入问题或填写信息，获得相应的校园生活建议和文本生成结果。

## 二、项目功能

### 1. 校园问答助手

用户可以输入校园生活相关问题，例如：

- 校园卡丢了怎么办？
- 图书馆可以自习吗？
- 快递在哪里取？
- 考试怎么复习？
- 宿舍灯坏了怎么办？

系统会根据关键词匹配相应回答，并给出校园生活建议。

### 2. 宿舍报修助手

用户输入宿舍楼号、寝室号和故障类型后，系统可以自动生成一段报修文本，方便学生向宿管或维修人员描述问题。

### 3. 课程学习建议

用户输入课程名称后，系统会根据课程关键词返回相应学习建议，例如程序设计、高等数学、大学英语等。

## 五、运行方式

本项目不需要安装额外环境，也不需要配置服务器。

运行步骤：

1. 下载或解压项目文件夹。
2. 找到 `index.html` 文件。
3. 双击 `index.html`。
4. 使用浏览器打开网页。
5. 在网页中输入问题或填写信息，即可测试项目功能。
